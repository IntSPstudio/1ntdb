From here you can access the tables directly but normally, the data is not accessed directly through tables. But through reports, forms, queries, etc.

About this structure. Database is divided into ten levels. Default is zero. These can be thought of in a way as generations or 'regimes'.

Every tables ID is four digit number: Database level number (0-9), followed by the table's own unique three-digit number. This three-digit number is formed as follows: 1. block, 2. group, 3. class.
